# Fetch Freight Mujoco

![freight robot](images/freight.png)

Mujoco model for the Fetch Robotics Freight Robot

## Prerequisites

Mujoco 

```bash
pip install mujoco
```

## View Model

```bash
python -m mujoco.viewer --mjcf xml/freight.xml
```
